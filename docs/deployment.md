# Deployment Guide (WIP)

This guide details how to deploy your own Ark using Ansible and Terraform.

## Prerequisites

- A remote server (VPS) with Root access.
- Basic knowledge of CLI.
- A compatible thin-client device.

## Steps

1. Provision the server.
2. Run the Ansible playbooks.
3. Connect your thin client.
